# Mkt Virtual Laravel template

## Instalação

`composer install`

`npm install`

`php artisan key:generate`

## Configuração

`.env.example` > `.env`

`php artisan migrate`

`php artisan db:seed`

## Rodando

`php artisan serve`

`npm run watch`

## Recursos
* PHP 7+ (tested on 7.1.1)
* MySQL 5.7+ (tested on 5.7.15)
* Laravel 5.6
* Admimbsb 1.0.5
